AMA with 我有饭创始人冯铮

发表于 2015-12-03   |   分类于 市场   |  

嘉宾介绍：冯铮

清华大学电子工程本科毕业后赴美国耶鲁大学完成生物统计硕士学位。在美期间创建微博大号“纽约吃货”，粉丝超过 20 万。回国后曾任职 IDG 担任投资经理。2015 年 1 月创办【我有饭】，目前已是中国最大私房饭局共享平台，成功举办饭局超千场。

精选问答

关于【我有饭】这个产品

@新君同學：我有饭 APP 的目前用户量，饭局数量，交易额如何？有目标吗？第二，现在平台没有收取费用，以后怎么盈利呢？

我们目前用户近 30w，饭局数量已经办过 1000 多场，交易额几百万了。目前我们的拼桌饭局是不抽成的，鼓励 host 多多报名，在平台上体验做 host 的感觉，未来会通过其它服务盈利。

@卓：APP 为什么用微信登录了 还要求注册呢？出于什么考虑

登陆问题我们也在调整，手机绑定主要考虑多平台同步账号信息的问题，另外我们有时间点的短信通知，也需要手机号。

@任意炸Sherry：最初 host 入驻平台的标准是怎样把关（我有饭本身是否把控）？未来是否可以 LBS 搜索附近饭局，共享经济下感觉我有饭很适合内容营销啊因为卖的不是饭是人嘛~

我们把关上门审核，而且都是高标准，初期选择有趣有特色的，lbs 也在这周更新的版本就会有了。

@小王：在北京饭局会不会成为和嘉宾见面的一种手段而已，想问，如果没有嘉宾，单纯冲着吃饭而去的用户多么？

嘉宾部分只是我们发掘一下饭局中的社交价值，但我相信饭局还是以美食为主的。大家不是仅仅为了见人，是因为东西好吃，还有有趣的话题才去的。比例上大部分是没有话题和嘉宾的，另外还有些嘉宾是 host 自己请的，比如我们一个 host，之前的饭局都是美食，上周的请到了金马奖最佳纪录片的导演周浩，也是饭局中的彩蛋吧。

【我有饭】跟竞品们的相爱相杀

@野田桑颠：昨天下了回家吃饭，满心以为是有闲的家人提供一些私家菜服务，自己还默默开了一家小馆，后来发现很多是小餐馆的外卖伪装成的，感情很受伤..

专车司机是不可能一个人同时开好几辆车的。。但是做饭这个事情是可以用小作坊来和掺和的。外卖用户看不到实际餐厅，这算一个模式的 bug 吧。

@香梨小金刚：hi，我有饭 有这么多竞争对手，你觉得你们突出在哪里？

现在我们饭局上的竞品很多，北京和上海的包括 xx、xxxx、xxxx、xx 等等。（为了和谐社会我们都和谐啦）

  • 1、我们是最早启动、规模最大、host 和用户都最多的平台，所以我们能够争取到最优质的 host、最有趣的食客群体，包括很多主题饭局，和 ikea、uber、创业公司 CEO 等等，给用户提供最有趣的体验。简单说就是俺们饭局的氛围和体验会比较好～
  • 2、我们一直坚持饭局销售不补贴（虽然销售难度很大），这样做的原因，是因为饭局不是简单的商品交易，而是有食客与主人、食客与食客之间的交流，我们希望通过饭局的价格筛选机制，能够为主人找到真正欣赏 ta 手艺的食客，也让参与的食客能够在饭局中认识有趣的人。说到底还是占了先发优势，作为市场上一直的第一名，是可以多想想如何给主人和食客谋福利的～～

【我有饭】的版图

@孙安：我有饭这种非标化的产品&服务，该如何复制和扩张呢？

我们觉得我有饭和 airbnb 很像，其实现在有手艺爱做饭的人很多，有餐饮梦的人也很多，我有饭最终其实是营造一个社区氛围，让这些有这方面想法的人，第一反应就是来我有饭做 host，这个才是最核心的方式。也就是说我们把社区氛围运营起来，让有意向的主人主动来申请，我们平台帮助他们实现自己的梦想。

@阿～Hong：我有饭针对的消费群体是哪些呢？主要的宣传传播途径是什么呢？

关于用户画像，我们的拼桌饭局上（陌生人饭局），主要是有一定消费能力，希望有新的消费能力，也原因认识新朋友的一群人，年龄在 20-35 之间，女生更多一些。而且大家都是非常有包容精神和分享精神的人，所以整个食客群的氛围特别好。另外我们最近还开辟了包桌服务，大家需要过生日、朋友聚会、请客吃饭，可以在平台上以包桌的形式和主人预定一场饭局。这一部分的受众人群就更广了，简单说，在我有饭寻找一种比餐厅更有趣的就餐体验。

我们现在主要的宣传途径就是微信公众号“我有饭”，我们的饭局信息会在这里发布，欢迎大家关注。我们也开发了 APP，搜索我有饭就可以，对于重度使用的用户，大家会逐渐迁移到上面来。因为我们的饭局体验很独特，所以口碑传播很多，大家吃完以后分享朋友圈的比例特别高。

@阿～Hong/@蔡璐璐/@kaidy：什么时候进驻深圳，武汉，青岛市场，好想体验！

我们希望尽快进入更多城市，包括国外和国内的，现在只要是要找到合适的城市经理人选，也欢迎大家自荐和推荐啊。未来逐步进入更多的城市。我们对城市的考虑，主要是看城市的美食氛围、也看城市特点，我们希望找 1. 大城市 2. 旅游城市 3. 国外华人多的城市

@蔡璐璐／@nathan：城市经理的工作主要是负责什么？招聘过程中遇到最大的困难是什么呢？你们在进入一个城市是否合适的标准是什么？只是城市经理么？还是会考核其他指标

一起回答吧，城市经历相当于城市的负责人，要负责开拓 host、饭局组织运营以及一些推广工作。核心来说，希望找到对美食有兴趣的人，也希望有专业的工作技能，美食的工作虽然有很多快乐，但是做起来也有很多困难

@野田桑颠：刚才提到的优势，比如启动早(新用户并不知道) 规模大(未可知) 优质 host 多(其它竞品的 host 也不赖) 无补贴(诚如你说带来销售难度) 人文关怀(这个好像确实是噢，评论做得蛮好的，正能量满满，时间合适很想参加一场。但整体与其它区别不明显) 以上都不算是无可替代的核心竞争力，boss 有没有找到更有力的突破点呢？诸如线上线下广告投入或者主题活动策划等等等等~~~

很中肯阿～刚提到了，我有饭和 airbnb 很像，虽然是一个交易平台，但本质上也是一个社区，对于社区来说，能够保持领先，汇聚最好的用户，实际上就是最重要的这个过程中我们要做的就是设立好规则，让更多的人参与近来。比如 host 吧，的确各个平台都会有，而且也不可能终身绑定，那么我们要做的就是给他们提供在我们平台上最好的体验（包括交易量和食客质量），让优质的 host 愿意留在我有饭这里。另外我们的主题饭局策划也是非常活跃的，刚刚也提到，有品牌、ceo、话题饭局、大v。另外就是对于 host 的开发，我们也在摸索能够批量找到有意向做 host 的人的渠道和方法，这个应该算是能够突破的地方，现在也积累了不少惊艳。

【@毛：具有话题性的主题活动，这个确实是很好的点~类似之前 uber 的一键叫 CEO】

中间大家问题太多，认真总裁总结了大家的问题 2333~~

大家的问题，1. 回家吃饭比较 2. enjoy 模式比较 3. 品类扩张（包括美食实验室）

饭事都是精品餐厅拉，他们基本没有私厨～～私厨的销售难度是最大的，我们先把 hard 模式做了～～
那就先说 enjoy／fancy 吧，其实我们和他们相比，我们针对的基本是类似的人群，就是有餐饮消费需求的人，而且私厨的销售要比餐厅销售难度还大。美食实验室最近为了饭局数量的考虑，也增加了很多餐厅的饭局。我们初期为了食客的体验考虑，是非常强调要去私房饭局，因为我们相信这个品类是我们真正能够提供独特价值的，而且聚集来的用户也是非常优质的。从我们用户需求的角度来说，未来合适的时候是可以上餐厅、商品这样的品类，让大家有更多的选择。简单的说，我们现在先通过私房饭局把 hard 模式过了，后面就容易了～
关于回家吃饭，的确他们切入了吃领域最高频的外卖，所以单量很容易做。
但是我觉得，平价的外卖盒饭是一个靠效率的事情，30 块钱以下的盒饭个人生产从效率上是否合理是个问题。平台如果追求交易量，最后是否会被那些经常被曝光的黑作坊占领，是一个需要考虑的问题。
我们现在做的饭局，包括美食活动和聚餐需求，虽然不如外卖频次高，但是本质上私房饭局是符合用户对消费体验的追求的，另外，美食的市场毕竟太大了，不算外卖，我们能够把餐厅里包厢和大桌的需求满足，也是非常大的市场了。我们现在复购率有 40%

吃饭这件大事

@蔡璐璐：请问我有饭有没有考虑过安全方面的问题，我想开私厨，租个地方对我来说好贵，在家我有担心安全。您之前做过一个20 万粉丝的微博号想跟您请教一下自媒体如何涨粉这种弱鸡鸡问题啊，求分享

我们在饭局上线前，会有同事实地考察，饭局也有评价机制，可以有效监督主人。另外我们的饭局都是去主人家，大家看着主人在厨房中操持，其实更放心啦。关于人身安全，第一我们现在都是实名支付手段，另外，饭局是个多人的环境，也是一个保障。至于一起做的微博 @纽约吃货，那的确是微博红利期间，在大家还没有开始发掘的时候先做了起来，后面就比较容易了。其实每一个平台成长中都会有一些公司在这个变化中获利，这就要运气＋准备了。**

@爱拍兔：有些人认为跟熟悉的人吃饭才有味道，不习惯跟陌生人吃饭，你觉得这样的人群也可以变为你们的用户么？还是说只针对自己的精准目标人群，不会考虑扩大受众人群？

你说的对，其实就是我们之前提了，有些人希望扩大自己的朋友圈，或者平常都是“一人食”很孤单，那么他们希望和大家一起吃，就是用我们的拼桌产品。我们发现拼桌是在组织一场美食活动，很多人非常喜欢。当然同样有人希望请朋友吃饭，我们现在也有包桌服务（目前可以在微信自定义菜单找，正在内测，近期会上线），大家就可以在平台寻找自己喜欢的私房主人，付款后请朋友一起去了。

另外还有个小功能即将上线，在 APP 内，大家以后可以看到自己朋友们报名了哪些拼桌饭局，就可以跟着一起报名了，这样也能解决想跟朋友一起去吃饭的问题。
