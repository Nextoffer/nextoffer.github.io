AMA with Flipboard 中国区负责人 Jing

发表于 2015-09-18   |   分类于 技术   |  

关于 AMA

AskMeAnything 是 Nextoffer 推出的一个互联网从业人员的问答节目。

AMA 每周会请来一位嘉宾，每位嘉宾都是在技术、产品、设计、运营等任意一个领域有着丰富经验、或是能力突出的大咖，通过一个小时与用户的交流与问答，为大家带来一些宝贵的职场经验与深刻的人生感悟。

嘉宾介绍：赵晶

Flipboard 中国区总经理, 负责 Flipboard 在中国业务。加入 Flipboard 之前，他是旧金山一家移动通讯应用公司 SocialDial 的联合创始人之一。之前任思科公司高级工程师。赵晶毕业于北京大学计算机系，并获得宾夕法尼亚州立大学博士学位, 研究移动计算。

  • 北京大学计算机系学士
  • 美国 Pennsylvania State University 计算机系博士
  • SocialDial (mobile group calling app, 美国旧金山) 联合创始人, 技术负责人
  • Flipboard 中国区总经理

精选问答

任意炸Sherry: Flipboard 九宫格的形式是谁想出来的？怎么想出来这么好的点子？

这是创始之初，CEO Mike McCue, designer Marcos Wescamp, 和 co-founder Evan Doll 共同想出的点子

阿左： Flipboard 是原生的还是用的 H5 canvas？

原生的

一名来自杭州（PM）Android版 2.3 版本的用户： 2.3 到 3.0 的改版初衷？首页增加的 Tab 栏是不是有点影响原有的浸入式阅读体验呢？

希望提供一个更简单的入口，让用户更快更直接的接触内容本身，在阅读内容过程中不断发现新的内容。另外两个那么做的原因是因为 1.) 当我们的推荐和个性化系统不断优化后，也有了信心能够给用户在一维的信息流里提供用户喜欢的内容，而更少依赖用户自己去主动选择内容 2.）更方便用户去发现新的好的内容，扩展新的兴趣点。因为我们发现会主动去添加新源，定制 Flipboard 内容的用户只是很少一部分

大鱼,一位北京的开发工程师: 每日精选是基于什么规则或算法抓取的，还是说是编辑人力选择的？另外互联网圈单身男士如何找女朋友？

算法先从每天几十万的文章中计算出热度高并且具备多样性的一个几十到一百的文章池子，然后编辑在做最终挑选的决定。 把你的 Flipboard 杂志做的更有趣，彰显个性，然后让我们把你推荐为 Flipboard 杂志达人，会有不少颜好才高志趣相投的女生关注的。

yao: 可以谈谈 Flipboard 的架构吗？

搭建修改了 4 年多的构架还是挺难很短时间内说清楚。现在几千台机器分布在 3 个大洲服务上亿用户. 非常欢迎你到 Flipboard 北京来看看我们，我们一定详细给你介绍。 Flipboard 最喜欢结交技术控的朋友！

luckyzhe: Flipboard 和 Zaker 区别在哪？你觉得Flipboard超越其他同类产品的点在哪里？

我们一直保持专注的做高质量，有价值内容的聚合和发现。我们相信小而美的产品能够找到足够多的喜欢我们的用户。我们不太急于为现金流而去尝试一些偏离我们基调的东西，比如 O2O 本地商户和消费者的互联。 基于这个理念，我们在中国目前的努力主要在通过技术和合作两方面为 Flipboard 接入更多好内容，通过技术和产品把这些好内容更有效的呈现给读者。这两方面我们过去一年有进步，但是离我们的目标还很远，所以我们很专注地继续做更好。我很佩服 Zaker 这一年来做的一些新方向的尝试，比如社区，比如本地玩乐。这可能会帮他们开拓出更广阔的市场，触达不同类型的用户，但这也是跟我们不太一样的地方。

任意炸Sherry 一位狂热的粉丝: 可以聊一下 Flipboard 接下来的发展战略吗？有说会被 Twitter 收购的？

Flipboard 在成立至今，一直都有各种被收购传闻，Twitter，Google, Yahoo 等等。这是对我们工作和产品的一种认可。但是收购当然不是我们的战略。我们的目标从未改变，就是要做全球最好的发现高品质，有价值内容的平台。 在收购了 Zite 后，我们也在努力搭建我们的个人兴趣图谱，通过个人兴趣，全球新闻内容覆盖，和高质量内容三方结合让这个平台释放更大能量。

msp的昌伟哥哥: Flipboard 目前有招聘计划吗？Flipboard 对人才有什么要求？

有有有！ 我们北京公司全线招。 技术方面 Java 后台平台，大数据平台，前端 web，iOS，安卓。产品经理急需。市场运营方面也急需。要求方面我觉得找得到这个群的都是有追求格调高的朋友。所以基本要求都过了。第二点就是你会认可我们产品和团队的理念 (以上提到的点)。并且你也喜欢类似硅谷，产品为中心的环境和工作方式。 剩下就是欢迎到 Flipboard 来看看，更深了解我们。

马业案一位来自北京的产品助理: 你觉得运营转产品，需要哪些准备？

做产品最重要是要喜欢这个产品，然后能一天花很多时间去琢磨设计，琢磨数据，去做探索来把这个产品变得让自己更喜欢。然后就是找到一个机会去实战学习。没有任何一种学习会比实际做学得更快。别担心没经验，一个有趣的事是如果你真的用心去喜欢和体验一个产品，深度去用，仔细思考，很快你会发现你比做这个产品公司里的很多员工都更了解这个产品。

无脸人: 你是如何努力做到了 Flipboard 中国区总经理及心路历程？

喜欢这个产品，所以加入了 Flipboard。努力工作把 Flipboard 变的让自己更喜欢。作为一个中国人我喜欢 Flipboard，也看到很多 Flipboard 让中国人不喜欢的问题，然后就努力去想一切有可能的办法去改变它，很多都是超过我当时职能范围应该
